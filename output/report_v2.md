## 핵심 코드 로직

### 노이즈 라인 필터 (신규)
```python
NOISE_PATTERNS = [
    r"^\d{2}-\d{4}-\d{7}$",   # 출원번호 단독 줄
    r"^\d+/\d+$",              # 페이지번호 (1/6)
    r"^수신\s*:.*$",
    r"^-\s*아\s*래\s*-$",
]
```

### 표 bbox 기반 content 중복 제거 (신규)
```python
line_bbox = line["bbox"]
in_table = any(
    lbbox[0] >= tb[0]-2 and lbbox[1] >= tb[1]-2 and
    lbbox[2] <= tb[2]+2 and lbbox[3] <= tb[3]+2
    for tb in page_bboxes
)
if in_table:
    continue  # 표 내부 텍스트 스킵
```

### 메타 정규식 개선 (공백 포함)
```python
# 출원인 (공백 포함 패턴)
re.search(r"출\s+원\s+인\s+성\s+명\s+(.+?)(?=주\s+소|\n대\s+리)", page1_text, re.DOTALL)

# 대리인
re.search(r"대\s+리\s+인\s+성\s+명\s+(\S+)", page1_text)

# 발명자 (반복)
re.findall(r"발\s+명\s+자\s+성\s+명\s+(\S+)", page1_text)
```

## 발견된 문제점

| # | 문제 | 원인 |
|---|---|---|
| 1 | 발명자/출원인/대리인 여전히 누락 | 1페이지 실제 구조 미파악 - `출\s+원\s+인\s+성\s+명` 패턴도 안 맞음 |
| 2 | 발명의명칭 누락 | 정규식 앵커 패턴 불일치 |
| 3 | 첨부 1개만 수집 | `re.findall` 패턴에 greedy 문제 |
