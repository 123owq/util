## 핵심 코드 로직

### 메타 추출 - 1페이지 특수 레이아웃 대응 (핵심 변경)

1페이지는 표 레이아웃으로 각 한자씩 별도 줄로 추출됨:
```
출       ← '출원인 성명' 레이블이 한 글자씩 분리
원
인
성
명한국타이어앤테크놀로지 주식회사   ← '명' 바로 뒤에 값이 붙음
주
소
```

대응 패턴:
```python
# '명XXX\n주' 패턴으로 사람 필드 일괄 추출
person_values = re.findall(r"명([^\n]+(?:\n[^\n]{5,})?)\n주", page1_text)
# → ['한국타이어앤테크놀로지 주식회사 (특허고객번호: ', '120120550993)', '한상수', '박해민', ...]

# 첫 번째: 출원인, 두 번째: 대리인, 나머지: 발명자
meta["출원인"] = cleaned_persons[0]
meta["대리인"] = cleaned_persons[1]
meta["발명자"] = [p for p in cleaned_persons[2:] if re.match(r"^[가-힣]{2,5}$", p)]

# 출원번호: '호 10-2023-0008170'
re.search(r"호\s+(10-\d{4}-\d{7})", page1_text)

# 출원일자: '자2023.01.19.'
re.search(r"자(\d{4}\.\d{2}\.\d{2}\.)", page1_text)

# 발명의명칭: '칭시트고무...\n정량화 방법'
re.search(r"칭(.+?)(?=\n발송번호)", page1_text, re.DOTALL)
```

### 비교표 구조화 (신규)
```python
def parse_comparison_table(raw_table):
    col_names = ["구성", "청구항제1항", "인용발명1", "비고"]
    rows = []
    for row in raw_table[1:]:   # 헤더 행 스킵
        entry = {col: row[i] for i, col in enumerate(col_names)}
        rows.append(entry)
    return {"원본_헤더": raw_table[0], "rows": rows}
```
